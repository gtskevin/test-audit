# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by
opening a [GitHub Security Advisory](https://github.com/gtskevin/test-audit/security/advisories/new).

Please do not report security vulnerabilities through public GitHub issues.

## Response Time

We aim to respond to security reports within 48 hours and provide a fix within
7 days for confirmed vulnerabilities.
